# Agent ops

## Culture

- Cite-or-cut: every subsystem maps to a primary source under `references/model-factory/`.
- Phase-gated memos after each stage gate.
- Agents propose; humans approve spend > USD 25 and any private upload.

## Standard memo sections

1. Hypothesis ID / stage
2. What changed
3. Metrics (with split name)
4. Cost
5. Decision (promote / iterate / kill)
6. Citations
