# On-policy distillation

## Primary sources

- TML OPD: https://thinkingmachines.ai/blog/on-policy-distillation/
- OPSD: https://siyan-zhao.github.io/blog/2026/opsd/

## Transfer boundary

- Reverse-KL OPD needs teacher logprobs on student tokens + token compatibility.
- Closed text APIs ⇒ sequence distillation, not OPD.
- Published OPD examples are often same-family (e.g. Qwen).
